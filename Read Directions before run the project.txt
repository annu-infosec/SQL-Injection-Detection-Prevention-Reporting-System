How to run the SQL Injection Detection, Prevention & Reporting System Project :-

1.Copy Web project (SQL DPR) folder and Paste inside root directory(for xampp xampp/htdocs)

2.Open PHPMyAdmin (http://localhost/phpmyadmin)

3.Create a database with name Web project (SQL DPR)

4.Import Web project (SQL DPR).sql file in database (given inside the folder)

5.Run the script http://localhost/Web project (SQL DPR)
